Total Weight Reckoner

Calculates Total Gross Weight and Total Net Weight

This module will calculate the total net weight and gross weight and it will be displayed on the bottom of the order lines / invoice lines in Sales Orders, Purchase Orders, Invoices and Quotations.

After installing the module you can easily map the fields to invoices, sales oders, purchase orders and quotations.

No configuration required. Download, install and start using the feature.
